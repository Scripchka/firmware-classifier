# IoT Firmware Security Classifier

Проект для классификации прошивок устройств IoT (Интернета вещей) на предмет безопасности. Приложение использует глубокое обучение для определения, является ли прошивка безопасной (benignware), взломанной (hackware) или вредоносной (malware).

## 🚀 Возможности

- **Графический интерфейс**: Удобный GUI на базе Tkinter для загрузки и анализа файлов
- **Классификация изображений**: Анализ изображений прошивок
- **Классификация бинарных файлов**: Автоматическое преобразование бинарных файлов в изображения для анализа
- **Модель EfficientNet**: Использование предобученной модели EfficientNet-B0 для высокой точности
- **Детальные результаты**: Подробный отчет с вероятностями и рекомендациями

## 🛠 Установка

1. **Клонируйте репозиторий:**
   ```bash
   git clone <repository-url>
   cd mp_proj
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Убедитесь, что модель находится в папке `model/`:**
   - Файл `efficientnet_best.pth` должен быть в директории `model/`

## 🎯 Использование

### Запуск приложения

```bash
python firmware_classif.py
```

### Интерфейс приложения

1. **Загрузка файла:**
   - Нажмите "📁 Загрузить изображение" для выбора изображения прошивки
   - Или "🔧 Загрузить бинарный файл" для выбора файла прошивки (.bin, .hex, .rom и т.д.)

2. **Предпросмотр:**
   - Для изображений: отображается миниатюра
   - Для бинарных файлов: создается визуализация на основе байтов

3. **Классификация:**
   - Нажмите "🎯 ЗАПУСТИТЬ КЛАССИФИКАЦИЮ"
   - Получите подробный отчет с вероятностями для каждого класса

### Классы классификации

- **✅ Benignware**: Безопасная прошивка
- **⚠️ Hackware**: Взломанная/модифицированная прошивка
- **❌ Malware**: Вредоносная прошивка

## 🧠 Обучение модели

Для обучения собственной модели используйте скрипт `model_train.py`:

```bash
python model_train.py
```

Скрипт использует данные из папки `data/iot_firmware/imagery/` и сохраняет лучшую модель в `efficientnet_best.pth`.

## 📋 Требования

- Python 3.7+
- PyTorch 1.9+
- TorchVision 0.10+
- Pillow 8.3+
- NumPy 1.21+
- Tkinter (обычно включен в Python)

## 🔧 Технические детали

- **Модель**: EfficientNet-B0
- **Количество классов**: 3 (benignware, hackware, malware)
- **Размер входа**: 224x224 пикселя
- **Предобработка**: Нормализация ImageNet
- **Для бинарных файлов**: Преобразование в grayscale изображение на основе байтов

## 📊 Результаты классификации

Приложение предоставляет:
- Вероятности для каждого класса
- Уровень уверенности (очень высокая/высокая/средняя/низкая/очень низкая)
- Визуальную шкалу вероятности
- Итоговый вердикт с рекомендациями

## ⚠️ Важные замечания

- Модель обучена на специфическом наборе данных IoT прошивок
- Для критически важных систем рекомендуется дополнительная верификация
- Точность зависит от качества обучающих данных
